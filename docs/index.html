О клиенте

Петровский Путевой Дворец - историческое сооружение, последний дворец на въезд в Москву из Петербурга. Дворец строился по приказу Екатерины II в связи с окончанием Русско-турецкой войны. Дворец имел стратегическое назначение, в нем останавливались императоры перед коронацией. Сегодня Петровский Путевой Дворец - это дом приемов правительства Москвы и отель.

Задача

Разработать корпоративный сайт для отеля «Петровский Путевой Дворец», с функционалом бронирования номеров через сайт, провести интеграцию с системой онлайн бронирования стороннего ресурса, подключения e-commerce на внутренний магазин дворца
